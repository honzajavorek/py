
# python.cz

Czech Python community homepage.

[![Build Status](https://travis-ci.org/honzajavorek/python.cz.svg?branch=master)](https://travis-ci.org/honzajavorek/python.cz)

## Installation

```shell
$ pip install -r requirements.txt
```

### Development

```shell
$ python runserver.py
```
