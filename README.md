
# python.cz

Small summary website about Czech Python community.

## Installation

```bash
$ pip install -r requirements.txt
```

### Development

```bash
$ python web.py
```

### Heroku

Read [this](https://devcenter.heroku.com/articles/getting-started-with-python).
There's ready-made `Procfile` in the repository.

### Production

```bash
$ gunicorn web:app
```
