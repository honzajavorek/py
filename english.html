Facts and Names

What is PyVo and Pyvec? This cheat sheet will help you with all the terminology:

Python CZ / @pythoncz
A portal for Czech Python programmers, and the effort to connect groups from different cities.
Pyvec / @pyvec
A legal entity founded to support Python in the Czech Republic.
PyVo in Brno, Prague, ... / @napyvo
Meetups in individual cities. The meetup in Prague was historically named Django CS.
Rubyslava + PyVo / @rubyslava
A group from Bratislava, Slovakia. They organize joint meetups with the Ruby community.
PyLadiesCZ / @pyladiescz
Czech chapter of an international community that aims to get more women into Python and programming in general.
