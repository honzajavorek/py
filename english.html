Start with Python

The Czech version of this page has a lot of information for Czech Python learners. English speakers can start with the official Python tutorial.

Get Involved

  • Discuss via e-mail, Facebook, or IRC. We usually speak Czech, but understand English as well.
  • Present at meetups! For 5-minute lightning talk, you can usually simply come and present. If you have a longer talk, please get in touch with meetup organizers.
  • Help open source projects! Many projects are looking for contributors at all levels. Github has a list of some of the more popular ones.

Organization

Pyvec is an organization aimed at helping the Czech Python community. It supports volunteers that help to advance Python, seeks sponsorship, and promotes various meetups, conferences and workshops.

Borrow a book

We have a library of Python-related books. If you want to borrow one, let us know, and come to the nearest Pyvo meetup to pick the book up. All the info is at the Bookshelf page.
